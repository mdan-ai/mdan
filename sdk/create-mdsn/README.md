# create-mdsn

`create-mdsn` scaffolds a minimal MDSN app.

Use it when you want the fastest path from a blank directory to a working MDSN app.

## Usage

```bash
npm create mdsn@latest agent-app
cd agent-app
npm install
npm start
```

The generated app uses this small shape:

- `app/index.md`
- `app/server.ts`
- `app/client.ts`
- `index.mjs`
