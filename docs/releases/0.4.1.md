# 0.4.1 Release Notes

## Bunx Starter Fix

`create-mdsn@0.4.1` is a focused patch release for the Bun starter path.

It fixes the CLI entrypoint behavior so `bunx create-mdsn ...` correctly executes the scaffold command when installed from the npm registry.

## Scope

This release only changes the published `create-mdsn` package behavior for Bun users.

- `@mdsnai/sdk` remains at `0.4.0`
- Node starter behavior is unchanged
- Generated starters still target the same shared app structure and runtime-specific templates introduced in `0.4.0`
