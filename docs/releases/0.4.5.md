# 0.4.5 Release Notes

## Current 0.4 Patch Release

This release updates the published package set to:

- `create-mdan@0.4.5`
- `@mdanai/sdk@0.4.2`

## What Changed

`@mdanai/sdk@0.4.2` includes the latest `auto` runtime work:

- `auto` is now an explicit operation-level keyword
- `auto` is resolved by the server host before returning the final result
- agent and browser consumers now see the same resolved result
- `label` is no longer used as an implicit execution signal

`create-mdan@0.4.5` stays on the `0.4` compatibility line and continues to generate starter projects against the compatible SDK minor range:

- `create-mdan 0.4.x` generates `@mdanai/sdk: ^0.4.0`
- the scaffold package itself no longer pins `@mdanai/sdk` as a direct runtime dependency

## Scope

- this is a patch release across the current `0.4` line
- no new starter runtime was introduced
- the main change is aligning host behavior, generated starter dependency strategy, and published package metadata
